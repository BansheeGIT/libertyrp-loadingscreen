# LibertyRP — загрузочный экран (loading screen)

Одностраничный экран подключения для GMod DarkRP-сервера LibertyRP.
Всё внутри одного файла **`index.html`** — шрифты (Oswald + JetBrains Mono с
кириллицей) и логотип уже вшиты как data-URI, внешних зависимостей нет.

## Что уже есть

- Оформление «ночной неоновый город», синие тона под логотип.
- **Сменяющийся фон** — генеративные сцены (полночь, дождь, снег, гроза, туман)
  с параллаксом, звёздами и погодой. Можно заменить своими картинками.
- **Музыка с автосменой треков** — плеер по кругу играет плейлист, кнопки
  вперёд/назад/пауза, эквалайзер, ползунок трека.
- **Прогресс подключения** — работает с настоящим GMod API
  (`GameDetails`, `SetFilesTotal`, `SetFilesNeeded`, `DownloadingFile`,
  `AllDownloadsComplete`), а вне игры крутит демо-загрузку.
- Ротатор советов/правил, панель с инфо о сервере, адаптивная вёрстка.

## Настройка

Откройте `index.html`, найдите вверху `<script>` блок **`CONFIG`**:

### 1. Музыка

```js
tracks:[
  {title:"Midnight Drive", artist:"Liberty FM", src:"https://cdn.libertyrp.ru/music/track1.mp3"},
  {title:"City of Freedom", artist:"Liberty FM", src:"https://cdn.libertyrp.ru/music/track2.mp3"},
],
```

- Ссылки — прямые на `.mp3` / `.ogg`, по **HTTPS**, с того же или CORS-разрешённого хоста.
- Треки играют по кругу с автопереключением; при `ended` включается следующий.

### 2. Свои фоны-кадры (вместо генеративных)

```js
backgrounds:[
  "https://cdn.libertyrp.ru/bg/city1.jpg",
  "https://cdn.libertyrp.ru/bg/city2.jpg",
  "https://cdn.libertyrp.ru/bg/city3.jpg",
],
```

Если массив не пустой — фон крутит эти картинки (плавный кросс-фейд каждые 7 c).
Пусто — рисуется генеративный город.

### 3. Данные сервера

```js
server:{ name:"LIBERTY RP", address:"play.libertyrp.ru:27015",
         map:"rp_liberty_city", gamemode:"DarkRP", maxplayers:128, ping:24 }
```

При реальном подключении GMod сам вызовет `GameDetails(...)` и перезапишет
имя сервера, карту, режим и слоты — эти поля можно оставить как есть.

## Подключение к серверу

1. Залейте папку `loadingscreen/` (достаточно `index.html`) на любой
   статический **HTTPS**-хостинг (ваш веб-сервер, GitHub Pages, Netlify и т.п.).
2. В `server.cfg` (или `cfg/`) пропишите:

   ```
   sv_loadingurl "https://ваш-домен/loadingscreen/index.html"
   ```

3. Перезапустите сервер. При заходе игрок увидит этот экран.

### Про автозвук в GMod

Загрузочный экран GMod работает на Chromium (CEF). В части сборок автозвук
разрешён — музыка стартует сама. Если он заблокирован, показывается плашка
**«Нажмите, чтобы войти»** — по клику звук включается. Кнопкой «Звук вкл/выкл»
в правом верхнем углу можно управлять звуком.

## Предпросмотр локально

`file://` в браузере блокируется политиками — поднимите локальный сервер:

```
cd loadingscreen
npx http-server -p 8080
```

и откройте `http://127.0.0.1:8080/index.html`.
